## Getting Started

### Prerequisites
- **iOS 26+**
- **Xcode 26+**
- **Swift 6+**

### Installation
1. Clone the repository:
```bash
git clone https://github.com/nacher8/WeatherSDP.git
```

2. Open the project in Xcode:
```bash
cd WeatherSDP
open WeatherSDP.xcodeproj
```

3. Build and run the project on your simulator or device.

## Technologies Used

- **SwiftUI** - Modern declarative UI framework
